Create Access Rights¶

Now create access rights for the Developer role.

Note

After a role is created there are no access rights defined in the role.

Edit Access Rights¶

Click on the Developer role to display the current access rights. Then click on the Edit Access Rights button.

Configure Access Rights¶

Now configure the desired access rights for the role. In this example custom access rights will be configured for the API Key resource.

Resource Action Qualifier
API Key Access MINE
API Key Create User Key MINE
  1. Click on the API Key checkbox and then the Custom Access radio button.
  2. Click on the Access checkbox and select the Mine qualifier.
  3. Click on the Create User Key checkbox and select the Mine qualifier.

Additional Access Rights¶

The access rights for the additional resources are created in the same way as was done in the step above for the API Key resource.

When finished click on the Save Changes button.
