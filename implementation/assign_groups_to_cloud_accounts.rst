Assign Groups to Cloud Accounts
-------------------------------

Okay, you have configured the Groups, Roles and Access Rights.  Now it's time to assign your Groups to Cloud Accounts. For each Group defined in the Company, you assign a Role to each cloud account that you want to allow the Group to access.
